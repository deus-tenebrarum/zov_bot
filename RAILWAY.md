# Railway: сохранить лидерборд и прогресс

Без Volume данные (лидерборд, прогресс) **стираются при каждом деплое**.

## Как добавить Volume (1 раз)

1. Открой [railway.app](https://railway.app) → свой проект
2. **Правый клик по пустому месту** (по серому фону, не по плитке сервиса) на странице проекта
3. В меню ищи **«Add Volume»** или **«Create Volume»**
4. Выбери свой сервис (приложение)
5. В **Mount Path** укажи: **`/app/data`**
6. Сохрани → Redeploy

---

**Вариант 2:** Кликни по **плитке своего сервиса** → откроется панель справа → вкладка **Settings** → внизу раздел **Volumes** → **Add Volume** (если есть).

**Вариант 3:** Кнопка **+ New** на странице проекта → в выпадающем списке ищи **Volume**.
