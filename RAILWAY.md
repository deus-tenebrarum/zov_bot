# Railway: сохранить лидерборд и прогресс

Без Volume данные (лидерборд, прогресс) **стираются при каждом деплое**.

## Как добавить Volume (1 раз)

1. Открой [railway.app](https://railway.app) → свой проект
2. Нажми **⌘K** (или Ctrl+K) → введи **Volume** → **Create Volume**
3. Выбери свой сервис (тот, что запускает приложение)
4. В **Mount Path** укажи: **`/app/data`**
5. Сохрани

После этого лидерборд и прогресс будут сохраняться между деплоями.
